    Network Classifications

  • Lan - A local area network that normally consists of a collection of computers in a single building or building complex.
  • Man - A metropolitan area network is a network of intermediate size, such as one spanning a local community.
  • Wan - A wide area network that links machines over a greater distance like neighboring cities or opposite sides of the world.

    • Network Models

  • bus network - A topology in which the machines are all connected to a common communication line called a bus.
  • star network - A topology in which one machine serves as a central focal point to which all the others are connected.

    • Protocols

  • CSMA/CD Ethernet - Carrier Sense, Multiple Access with Collision Detection is the protocol that controls the right to transmit messages and dictates each message be broadcast to all the machines on the bus.
  • CSMA/CA Wifi - Carrier Sense, Multiple Access with Collision Avoidance is usually standardized by IEEE within the protocols of IEEE 802.11, commonly referred to as Wi-Fi, and is the protocol designed to avoid collisions but may not eliminate them completely.

    • Combining networks:

  • repeater - A device that passes signals back and forth between the two original buses, usually with some form of amplification, without considering the meaning of the signals.
  • bridge - A device that connects two buses and allows the two machines residing on the same side of the bridge to exchange messages without interfering with communications taking place on the other side.
  • switch - A device that produces a network consisting of several buses extending from the switch.
  • router - A device which is a special purpose computer used for forwarding messages.
  • gateway - The "point" at which one network is linked to an internet. It serves as a passageway between the network and the outside world.

    • Client/server model

  • p2p: peer-to-peer - A client/server model that involves one process (the server) providing a service to numerous others (clients). The peer-to-peer model involves processes that provide service to and receive service from each other.
  • distributed system - Systems that consist of software units that execute as processes on different computers. This includes many modern software systems, such as global information retrieval systems, company-wide accounting and inventory systems, computer games, and even the software that controls a network's infrastructure.
  • cloud computing - A distributed system where huge pools of shared computers on the network can be allocated for use by clients as needed.

    • Internet Addressing

  • Tier 1 - ISPs that consist of very high-speed, high-capacity, international WANs. They are considered the backbone of the Internet.
  • Tier 2 - Tier-2 ISPs connect to tier 1 ISPs and tend to be more regional in scope and less potent in their capabilities. Tier-1 and tier-2 ISPs are essentially networks of routers that collectively provide the Internet's communication infrastructure.
  • Tier 3 - Tier-3 ISPs provide access to the most or all of the network that remains connected even when very important tier-1 routers fail.

    • IP address

  • ICANN - Internet Corporation for Assigned Names and Numbers is a nonprofit corporation established to coordinate the Internet's operation and is responsible for awarding blocks of consecutively numbered IP addresses to ISPs.
  • DNS - Domain Name System is a directory system that uses name servers to convert a mnemonic address into an IP address before transmitting messages.
  • TLD: Top-level domain - Top-level domains are the suffixes used to reflect a domain's classification. These include .edu for educational, .com for commercial institutions, .gov for U.S. government institutions, .org for nonprofit organizations, .mil for military, and .net which was originally intended for ISPs but is now used on a much broader scale.
  • Country-code TLDS - A top-level domain that uses two-letter TLDs for each specific country, such as .au for Australia and .ca for Canada. Furthermore, hundreds of TLDs were added to the DNS hierarchy, such as .biz for businesses and .museum for museums.

    • Internet protocols

  • FTP - File Transfer Protocol is an application used for listing and copying files across the network.
  • HTTP - Hyper Text Transfer Protocol is a webpage that handles the traditional network applications, such as the Network News Transfer Protocol (NNTP), File Transfer Protocol (FTP), and the Telnet protocol, or later, the Secure Shell (SSH) protocol.
  • SMTP - Simple Mail Transfer Protocol defines a way that two computers on the network may interact when transmitting an email message from one host to the other.

    • World Wide Web

  • URL - Uniform Resource Locator is a unique address given to the end of each document that contains the information needed by a browser to contact the proper server and request the desired document.
  • client-side activities - Activities that are performed by a client such as a browser.
  • server-side activities - Activities that are performed by a server such as a webserver.

    • Internet four-layer software

  • application layer - The application layer is where messages usually originate and consists of software units such as clients and servers that use Internet communication to carry out their tasks.
  • transport layer - The transport layer accepts messages from the application layer, ensures that the messages are properly formatted for transmission over the Internet, divides long messages into small segments called 'packets', and transmits packets across the Internet one segment at a time to the network layer.
  • network layer - The network layer is in charge of maintaining the router's forwarding table and using that table to determine the direction to forward packets.
  • link layer - The link layer is charged with receiving and transmitting packets and must deal with the communication details particular to the individual network in which the computer resides.

    • Internet is:

  • packet-switching network - Packet switching transmits data across digital networks by breaking it down into blocks or packets for more efficient transfer using various network devices.
  • fault tolerant - Fault tolerant refers to the ability of a system to continue operating without interruption.
  • redundant - The process of providing multiple paths for traffic so that data can keep flowing even in the event of a failure.

    • Cybersecurity

  • malware - Malicious software that transfers to or executes on the computer itself or attacks the computer from a distance. Examples of this software include viruses, worms, Trojan horses, and spyware, whose names reflect the primary characteristic of the software.
  • virus - A virus is a software that infects a computer by inserting itself into programs that already reside in the machine.
  • worm - A worm is an autonomous program that transfers itself through a network, taking up residence in computers, and forwarding copies of itself to other computers.
  • Trojan horse - A Trojan horse is a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim; however, it performs additional activities that might have harmful effects.
  • spyware - A malicious software, sometimes called sniffing software, that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
  • phishing - Phishing is a technique of obtaining information explicitly by simply asking for it, usually by an email posing as a financial institution, a government bureau, or perhaps a law enforcement agency.
  • DDos attack - A Dos or 'Denial of services' attack is an attack on a computer through other computers in the system, which is also called a 'bonnet' and a DDos or 'Distributed Denial of services' is a Dos attack generated over many hosts.

    • Protection

  • firewall - A program that filters traffic passing through a point in the network, which can help prevent unauthorized access to private data.
  • spam filter - A spam filter is a type of firewall designed to block unwanted emails.
  • proxy server - A software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • antivirus software - Software that represents a broad class of software products that are used as a means of defense against invasions via network connections and are designed to detect and remove the presence of unknown viruses and other infections.
  • cryptography - Cryptography is the study of sending and receiving messages securely in the presence of adversaries. Cryptographic tools such as 'encryption' can be used to transform a message into a coded version such that even if the encrypted data falls into unscrupulous hands, the encoded information will remain confidential.
  • public key encryption - The public key encryption is a system that uses two keys: the 'public key' which is used to encrypt messages and the 'private key' which is required to decrypt messages. This system involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
